# Oraksoft TW CSS Library

Bu proje, npm ile yüklenebilen basit bir CSS kütüphanesidir.

## Kurulum

```bash
npm install oraksoft-tw-css-lib
```

## Kullanım

CSS dosyanızı projenize dahil edin:

```css
@import 'oraksoft-tw-css-lib';
```

Veya HTML'de:

```html
<link rel="stylesheet" href="node_modules/oraksoft-tw-css-lib/dist/oraksoft-tw-lib.css">
```

## Geliştirme

Kaynak kodları `src/` klasöründe bulunur. Değişiklik yaptıktan sonra:

```bash
npm run build
```

## Yayınlama

Paketi npm'e yayınlamak için:

```bash
npm publish
```
